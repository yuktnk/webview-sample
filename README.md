# webview-sample

WebViewアプリのフルリプレイスPoCプロジェクトの実装サンプル。

## クイックスタート

```bash
nvm use          # Node.js バージョンを合わせる
pnpm install     # 依存パッケージインストール
pnpm local       # 開発サーバー起動（MSW自動起動）
```
